# Documentación de QA

Esta carpeta contiene la documentación y los artefactos relacionados con el proceso de testing del proyecto Booker.

Contenido:

- test-plan → estrategia y alcance de las pruebas
- test-cases → casos de prueba funcionales documentados
- bug-reports → registro de errores encontrados durante el testing
- reports → reportes generados automáticamente por las herramientas de testing (Supertest y Playwright)
